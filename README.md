# Android Project 4 - *Flixster+ Part 2*

Submitted by: **Irania Mazariegos**

**Flixster+ Part 2** is a movie browsing app that allows users to browse TV shows airing today. When the show is clicked, the rating, origincal language, and overview will be displayed.
